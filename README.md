# Iceberg3.1
迫不得已 我开源了冰山远程控制软件3.1

本来是不想开源的，但因为我的源码遭到了远控者的恶意盗窃，迫不得已把源码进行公开。请补药用于非法用途，欢迎吐槽我的代码。

功能介绍
基于易语言编写的远程控制软件，主要功能为远程屏幕/终端/文件管理等
使用tcp协议进行交互
